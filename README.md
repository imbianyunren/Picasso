# 養成遊戲之叫我畢卡索

這個遊戲能夠把你所畫的圖

利用AI去背，將你上傳的圖案變成遊戲主角

並可以餵他吃飯、帶他旅遊、工作，看他睡覺

陪著他一起長大😆

## 開發環境/安裝需求

* 開發環境：
  Develop Environment：android studio
  
  Create Virtual Device：Phone -> Pixel 4
  
  System Image：R，API Level 30，ABI：x86

* 安裝環境需求：

  android 11 以上(API 30)
